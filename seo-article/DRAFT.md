# DRAFT.md — 骨子生成・骨子レギュレーションチェック

STEP 4〜5 の詳細手順。

---

## STEP 4: 骨子生成

STEP 1〜3 の情報を統合して骨子を生成する。

### 4-1. メタ情報の生成

| 項目 | 生成ルール |
|-----|-----------|
| タイトル（H1） | KWを冒頭に含む・クリック率重視・32文字以内 |
| メタディスクリプション | KWを含む・120文字以内・読者へのベネフィットとCTAを含む |
| 想定文字数 | レギュレーション範囲内で競合平均を参考に設定 |
| ターゲット読者 | STEP 2の分析結果 |
| 検索意図 | STEP 2の分析結果 |

### 4-2. 見出し構成の生成

**原則**
- レギュレーションのH2・H3数を厳守
- 競合共通トピックを必ず網羅
- 差別化ポイントを1〜2箇所盛り込む
- 禁止語を使用しない
- 検索意図を満たす流れで構成する
- ターゲット読者の悩み → 解決 → 納得の流れを意識する

**見出しのKW配置**
- H1・最初のH2にメインKWを必ず含める
- 共起語・LSIキーワードを各H2に自然に分散させる

### 4-3. E-E-A-T補強ポイントの設定

各H2セクションに対して補強要素を注記する：

| 補強タイプ | 内容 |
|-----------|------|
| データ引用 | 統計・調査結果を引用すべき箇所 |
| 専門説明 | 専門用語の説明が必要な箇所 |
| 注意事項 | 免責・リスク説明が必要な箇所 |
| 外部リンク | 信頼性の高い外部リンクを貼るべき箇所 |
| 体験・実績 | 実体験・実績・事例を加えるべき箇所 |

### 4-4. FAQセクションの生成

検索意図から想定される「よくある質問」を3〜5個生成し、末尾のH2として配置する。

### 4-5. 内部リンク挿入指示

インプットの内部リンク先リストから適切な挿入箇所を選定し、各セクションに挿入指示を記載する。レギュレーションの内部リンク上限を超えないこと。

---

## STEP 5: 骨子レギュレーションチェック

生成した骨子を以下の基準でチェックし、問題があれば自動修正する。

| チェック項目 | 基準 |
|------------|------|
| KW配置 | タイトル・H1・最初のH2・メタディスクリプションにKWが含まれるか |
| 見出し数 | H2・H3数がレギュレーションの範囲内か |
| 禁止語 | 見出し・メタ情報に禁止語が含まれていないか |
| 文字数目安 | 想定文字数がレギュレーションの範囲内か |
| E-E-A-T | 各H2に補強ポイントが最低1つあるか |
| 内部リンク | 内部リンク指示数がレギュレーション上限以内か |

チェック結果を以下の形式で出力する（OK / 修正済み / 要確認 の3段階）：

```
【骨子レギュレーションチェック結果】
KW配置:    ✅ OK
見出し数:  ✅ OK（H2: {n}個 / H3: {n}個）
禁止語:    ✅ OK
文字数目安: ✅ OK（目安{n}字）
E-E-A-T:  ✅ OK
内部リンク: ✅ OK（{n}本）
```
