# AI驚き屋 マルチ発表版 Xポスト生成

複数の発表・アップデートが含まれるAIニュース記事を受け取り、
発表ごとにツリーを組んだ長尺Xポストシリーズを生成する。

## 入力

URLだけでよい。例：`/odoroki-multi https://example.com/ai-news`

## Step 1：URLからコンテンツ取得

WebFetchでURLの内容を取得する。抽出すべき情報：
- 記事内に含まれる全発表・アップデートの一覧
- 各発表の機能・できること・対象ユーザー・リリース状況
- 開発元・技術仕様・価格
- 競合・比較情報

403ブロックの場合はSocialData APIで検索にフォールバックする。

## Step 2：副業関連発表を最大3つ選ぶ

取得した全発表の中から **AI副業・業務効率化に関係するもの** を最大3つ選ぶ。

**選定基準（優先順）：**
1. AI副業・業務効率化への直結度が高いもの
2. ふりーのフォロワーが「使ってみたい」と思うもの
3. Antigravity / Claude Code / OpenClaw / Codexと比較できるもの

**ターゲットペルソナ：**
- ① AIで楽稼ぎしたいと思っている人
- ② AI副業をやっていてもっと効率的な使い方を知りたい人

**絶対に言及しない：**
- Web3・NFT・ブロックチェーン関連
- SEO・アフィリエイト・サイト運営
- 副業に関係ない機能（旅行計画・家の管理・ゲーム等）

## Step 3：比較ツールの判断

記事内容に応じて、以下のツールと比較するかどうか判断する。
比較するなら「セロイ投」に差し込む。

**比較対象として使えるツール：**
Claude Code / Antigravity / OpenClaw / Gemini / Claude / ChatGPT / Codex

**Antigravityへの言及がある場合：**
ふりーはAntigravityに詳しい人的な印象を持たれている。
Antigravityが絡む場合は「Antigravityユーザーとしての視点」で愚痴・考察を入れる。

## Step 4：ポスト構成を組む

以下の順番で生成する。長くなるのは推奨。CTAは入れない。

---

### 1投目：半熟型フック（全体イントロ）

**型：** 今日の発表全体への驚き・混乱・興奮を素直に出す

- 「うおおおおお！！！」「おいおい…」「マジか」「嘘やろ」系
- 選んだ3発表のタイトルをざっくり並べて「全部やばい」感を出す
- 結論は言わず、次のツリーへ引っ張る（**ツァイガルニク法**）
- 自分のことは「ワイ」で統一

---

### 発表①ツリー（2投）

**①-1投目：抽象（概要・箇条書き）**
- 発表名を冒頭に出す（**オロナミンC法**：固有名詞を使う）
- できること箇条書き（AI副業関連のみ、多めに）
- スペック情報も入れる（速度・価格・対応環境等）
- 最後の1行を途中で切って次の投稿へ引っ張る（**ツァイガルニク法**）

**①-2投目：具体（チェックリスト）**
- 「[企業]が公式に言ってる使い方↓」で始める（**マッキンゼー構文**）
- ✅ 〇〇を△△したら□□できる、の形で具体的に
- 「実際の操作・シーン」レベルまで落とす
- 最後に1行まとめ（「要は〇〇の感覚」）
- 🔥や😎などの絵文字を1個だけ末尾に使う

---

### 発表②ツリー（2投）

**②-1投目：抽象**（①-1投目と同じ型）
**②-2投目：具体**（①-2投目と同じ型）

---

### 発表③ツリー（2投）

**③-1投目：抽象**（①-1投目と同じ型）
**③-2投目：具体**（①-2投目と同じ型）

---

### ふりー独自投：ビフォアフ（全発表まとめ）

**型：** 今回の発表群を横断したビフォアフ。「実際に使うだろうな」と思うアングルだけ使う。全部使わなくていい。

**使えるアングル（記事に合わせて取捨選択）：**

**【ウラーカ（裏アカ運用・インプ稼ぎしてる人）】**
- Before：手動でやってた投稿・リサーチ作業
- After：今回の発表でどう自動化・高速化されるか

**【ブロガー】**
- Before：記事作成・リサーチにかけてた時間・手間
- After：今回の発表でどう変わるか

**【ワイ（ふりー自身）】**
- 「ワイはこう使うかもしれない」くらいの温度感で書く
- 断言しない。「〜に使えそう」「〜が変わるかも」くらい

最後に1行：「考える以外の作業を全部渡せる仕組みが、またレベル上がった」系

---

### セロイ投：逆張り・クソかも

**型：** 期待半分不安半分。ボロクソに言いつつ方向性は認める。

- 今回の発表群の制約・課題・怪しい点を正直に書く
- 比較対象ツールがあれば差し込む
- Antigravityが絡む場合：愛憎込みで語る（下記口調データ参照）
- 「逆張り＋根拠」で締める（**逆張り法**）
- ラスト：「期待はしてる。でも全ベットするのは〇〇になってから」系
- 😎で締める

---

### まとめ投：締め＋出典

**型：** 今日の3本を1行ずつ振り返って締める

```
今日まとめた発表↓
・[発表①名]：[一言]
・[発表②名]：[一言]
・[発表③名]：[一言]

元記事：[URL]
```

---

## 出力フォーマット

チャットに直接出力する。各投稿を以下の形式で区切る：

```
---
1/N（役割）
[本文]

---
2/N（発表①・抽象）
[本文]

...
```

Nは総投稿数（発表が3つなら合計10投）。

---

## ふりーの口調データ

Xポストデータから抽出した実際の文体。これを口調の基準にする。

**感嘆・フック系：**
- 「うおおおおお！！！」「おいおい…」「マジか」「嘘やろ」
- 「〇〇してたらこんなことになってた🔥」
- 驚きは大げさに表現する。「！！！」多め。

**自称・人称：**
- 自分のことは「ワイ」または「自分」
- 読者への呼びかけは「〜ですよ」「〜ですよね」「〜じゃないすか」

**語尾・語感：**
- 「〜ですね」「〜よな」「〜なんやが」「〜らしい」「〜ｗ」「笑」
- 関西弁と敬体が自然に混在する
- 「〜なんすよね」「〜でしかないんで」「〜すね」系の砕けた敬体

**絵文字の使い方：**
- 🔥：熱量・頑張り・良いニュース（多用OK）
- 😭：感動・感謝・嬉しさ
- 😎：強がり・自信
- 🥺：しんみり・困り・お願い
- 1投稿に1〜2個まで。多用しない。

**Antigravity語りの口調（特有）：**
- 「メンヘラばりに依存してる」
- 「DVの彼氏みたい」「だんだん本性出してくる」
- 「優しいと思ってたのに」系の愛憎表現
- 具体的な不満：「即トークン上限」「頻繁にサーバーエラー」「突然大改悪」「公式X燃えてるのにガン無視」

**実況中継スタイル：**
- 「〜してたらこうなった」「〜を試したら〜だった」系
- 自分の体験・発見をそのまま話す感じ
- 「ちょっとだけ分かったことがある」くらいのトーン（上から教えない）

**禁止：**
- 「保存推奨」「これを知らないと損します」「99%の人が知らない」
- 「〜について解説します」（AI感が出る）
- 対比構文の多用

## 注意事項

- 口調はふりーの実際の文体に合わせる（上記データ参照）
- 固有名詞は積極的に使う（**オロナミンC法**）
- 数字は具体的に書く（「数時間」より「週3〜5時間」）（**573法**）
- 長くなることを恐れない。むしろ長いほうが推奨。
- AI感が出ないよう「〜について解説します」はNG
